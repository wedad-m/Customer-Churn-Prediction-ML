# Customer Churn Prediction - Machine Learning

## Dataset Description
The dataset used in this project is the Telco Customer Churn dataset obtained from Kaggle. It contains 7,043 customer records with 21 features related to customer demographics, account information, and subscribed services.

## Machine Learning Problem
The objective of this project is to predict whether a customer will churn (leave the company) based on their service usage and account details.

## Problem Type
Supervised Classification

## Target Variable
Churn (Yes / No)
